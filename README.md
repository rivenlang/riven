# 📖 About

Riven is a modern compiled programming language built completely from scratch.

The long-term vision of Riven is to combine:

- ⚡ Native Performance
- 🤖 AI Development
- 🔄 Automation
- 🖥️ System Programming
- 📦 Modern Tooling
- 🧠 Clean Language Design

Unlike many projects, the goal of Riven is to understand and build every part of a compiler instead of relying on an existing compiler infrastructure.
